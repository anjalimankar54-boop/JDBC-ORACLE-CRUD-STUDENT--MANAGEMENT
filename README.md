##JDBC Oracle CRUD Student Management System
Project Overview
This project is a Java-based Student Management System developed using JDBC and Oracle 10g XE Database.
The main objective of this project is to demonstrate how Java applications interact with a relational database to perform complete CRUD operations (Create, Read, Update, Delete) in a real-world scenario.
The project is designed as part of a Junior Java Developer Internship to gain hands-on experience with database connectivity, SQL query execution, and exception handling using JDBC.


###Technologies Used
Java (JDK)
Oracle 10g XE Database
JDBC (Oracle JDBC Driver)
IntelliJ IDEA
GitHub
